A customizable Coming Soon/Maintenance Mode plugin for WordPress that allows you to display a professional coming soon or under-construction page.

== Description ==

**Simple Custom Coming Soon/Maintenance Mode** is a user-friendly WordPress plugin that allows you to create and customize a coming soon or maintenance mode page. Whether you're launching a new website or performing updates on an existing one, this plugin ensures your visitors are informed with a professional-looking page.

Features include:
- Easy setup with no coding required.
- Customizable messages to personalize your page.
- Lightweight and compatible with most themes.
- Responsive design to ensure your page looks great on all devices.

Keep your visitors informed while you work on your website behind the scenes!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/simple-custom-coming-soon` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to **Settings > Simple Custom Coming Soon** to configure the plugin and enable Coming Soon or Maintenance Mode.

== Frequently Asked Questions ==

= How do I enable the coming soon mode? =  
Go to **Settings > Simple Custom Coming Soon** and check the "Enable Coming Soon Mode" option. Save your settings, and the coming soon page will be active.

= Can I customize the coming soon page content? =  
Yes, you can customize the coming soon page by adding your own custom message in the settings page.

= Will the coming soon page affect logged-in users? =  
No, the coming soon page is only displayed to non-logged-in visitors. Administrators and logged-in users will still have full access to the site.

= Is this plugin compatible with any theme? =  
Yes, Simple Custom Coming Soon/Maintenance Mode works with any WordPress theme.
